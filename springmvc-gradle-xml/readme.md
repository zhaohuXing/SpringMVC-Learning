###springmvc学习
使用gradle进行构建，搭建基于xml的springmvc框架并配置JSP模板。分享个[Ubuntu下简易Javaweb开发环境](http://www.jianshu.com/p/0bbf593e3ecf),其次不使用idea开发javaweb,需要手动创建工程目录，编写了InitializeProject.sh进行自动创建，`./InitializeProject.sh 工程名`
<br />
